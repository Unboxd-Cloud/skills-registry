## Phase 1: Foundation

- Prepare nodes and baseline OS configuration
- Define network ranges, DNS, NTP, and certificate prerequisites
- Validate storage devices and replication expectations
- Record rollback point

## Phase 2: Cluster Bootstrap

- Initialize MicroCloud cluster
- Join nodes and verify cluster health
- Configure storage and network primitives
- Record rollback point

## Phase 3: Platform Services

- Install observability components
- Configure secrets and certificate flows
- Establish backup targets and restore verification
- Record rollback point

## Phase 4: Workload Enablement

- Define namespaces, projects, or tenancy boundaries
- Configure deployment pipelines
- Publish base images or artifacts
- Deploy first representative workload
- Record rollback point

## Phase 5: Readiness Review

- Run failure tests
- Run restore test
- Review dashboards and alerts
- Confirm ownership and on-call path
- Sign off against success criteria
