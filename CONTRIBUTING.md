# Contributing

## Scope

This repository publishes the Canonical MicroCloud Codex skill and its supporting references and templates.

## Change Rules

- Keep `SKILL.md` aligned with the published references and templates.
- Prefer additive changes for new migration patterns, integrations, or runtime guidance.
- Keep bundled templates operational and reviewable.
- Preserve separation between the standalone Agent ID protocol repo and this skill repo.
- Document any breaking structural change in `README.md` or `docs/README.md`.

## Quality Expectations

- Keep instructions concise and execution-oriented.
- Keep source-platform migration guidance explicit by platform.
- Keep runtime and protocol guidance scoped to clear boundaries.
